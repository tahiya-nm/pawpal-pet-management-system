# PawPal+ — Claude Guidance

## Core User Actions

These are the three primary things a user should be able to do in PawPal+:

1. **Add a pet** — Enter owner and pet info (name, species, and available time per day) to personalize the care plan.
2. **Add and manage care tasks** — Create tasks like walks, feeding, medications, or grooming, each with a duration (in minutes) and a priority level.
3. **Generate and view today's plan** — Produce a prioritized daily schedule based on available time and task priorities, with a plain-language explanation of why each task was included or skipped.
